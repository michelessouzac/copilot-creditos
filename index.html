Passo a Passo

Ativação do SE Copilot

Selecione o cliente e a operação Gerar Contrato/Licença. Certifique-se de marcar os campos Contratos e Licenças SE Suite.

Na aba de contratos, utilize o botão de adição (+) para criar a entrada de créditos de IA para o cliente.

Ajuste o Template para Usage e o Type para AI Credits. Defina a quantidade e valide o período de vigência.

Em Licença, altere para Renewal/Replacement e marque o campo Temporary License para finalizar o processo.
